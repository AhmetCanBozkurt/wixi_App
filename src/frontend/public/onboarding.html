01 — Vizyon

Wixi nedir?

Türk KOBİ'lerine hitap eden modüler SaaS platformu. Müşteri ihtiyacına göre modül seçiyor, kendi veritabanına sahip oluyor.

🏢

Multi-Tenant SaaS

Her müşteri (tenant) kendi izole veritabanına sahip. Veriler hiçbir zaman karışmaz. Tenant DB provisioning kayıt anında otomatik yapılır.

🧩

Modüler Yapı

E-Ticaret, Finans, Web Builder, CRM, HR gibi modüller bağımsız .NET projeleri. Müşteri sadece satın aldığı modülleri kullanır.

🏪

Storefront

Her E-Ticaret tenant'ı kendi müşterilerine açık online mağaza alır. Tema editörü ile drag-drop görsel özelleştirme yapabilirler.

Command — Yazma İşlemi
public record CreateProductCommand(
    string Name, decimal Price)
    : IRequest<Guid>;

public class CreateProductCommandHandler(ECommerceDbContext db)
    : IRequestHandler<CreateProductCommand, Guid>
{
  public async Task<Guid> Handle(
      CreateProductCommand req, CancellationToken ct)
  {
    var p = new WixiProduct {
      Id = Guid.NewGuid(), Name = req.Name, Price = req.Price
    };
    db.Products.Add(p);
    await db.SaveChangesAsync(ct);
    return p.Id;
  }
}
Query — Okuma İşlemi
public record GetProductsQuery()
    : IRequest<List<ProductDto>>;

public class GetProductsQueryHandler(ECommerceDbContext db)
    : IRequestHandler<GetProductsQuery, List<ProductDto>>
{
  public async Task<List<ProductDto>> Handle(
      GetProductsQuery req, CancellationToken ct)
  {
    return await db.Products
      .Where(p => !p.IsDeleted)
      .Select(p => new ProductDto(p.Id, p.Name, p.Price))
      .ToListAsync(ct);
  }
}

// CRMModuleExtensions.cs
public static class CRMModuleExtensions
{
    public static IServiceCollection AddCRMModule(
        this IServiceCollection services, string connectionString)
    {
        services.AddDbContext<CRMDbContext>(o => o.UseSqlServer(connectionString));
        services.AddMediatR(cfg => cfg.RegisterServicesFromAssembly(typeof(CRMModuleExtensions).Assembly));
        return services;
    }
}

public class RegisterUserCommandHandler(IMailQueue mailQueue, ...)
{
    public async Task<Guid> Handle(RegisterUserCommand req, CancellationToken ct)
    {
        // ... kullanıcı oluşturma ...

        mailQueue.Enqueue(new MailMessage
        {
            To           = req.Email,

13 — Debug

Sık Karşılaşılan Hatalar

Geliştirme sırasında karşılaşabileceğin yaygın sorunlar ve çözümleri.

❌ 401 Unauthorized her istekte

Token süresi dolmuş veya localStorage'da yok.

→ Tarayıcı konsolunda localStorage.getItem('token') kontrol et
→ axiosConfig.ts'deki interceptor refresh endpoint'i çağırıyor mu log'a bak
→ Backend'de JWT SecretKey appsettings.json ile eşleşiyor mu?

❌ Migration hatası — tablo zaten var

Farklı modülde aynı tablo adı varsa çakışır.

migrationBuilder.Sql("""
IF OBJECT_ID(N'TABLO', N'U') IS NULL
BEGIN
    CREATE TABLE [TABLO] (...)
END
""");

❌ CORS hatası (preflight blocked)

Backend CORS politikası frontend'i blokluyor.

→ Program.cs'de SetIsOriginAllowed(_ => true) aktif mi?
→ AllowCredentials() + AllowAnyHeader() + AllowAnyMethod() var mı?

❌ Tenant verisi gelmiyor

X-Tenant-Slug header eksik.

→ localStorage['wixi-active-tenant'] dolu mu?
→ axiosConfig interceptor header'ı set ediyor mu?
→ TenantMiddleware header'ı okuyup context'e set ediyor mu?

❌ EF DbUpdateConcurrencyException

Aynı kayda eş zamanlı yazma.

try { await db.SaveChangesAsync(); }
catch (DbUpdateConcurrencyException ex)
{
    var entry = ex.Entries.First();
    await entry.ReloadAsync();
    await db.SaveChangesAsync(); // retry
}

❌ TypeScript tipi hatalı — AdvancedDataTable

Interface Record constraint eksik.

// YANLIŞ:
interface ProductDto { id: string; name: string; }

// DOĞRU:
interface ProductDto extends Record<string, unknown> {
  id: string; name: string;
}
