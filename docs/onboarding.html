01 — Vizyon

Wixi nedir, ne yapmaya çalışıyor?

Türk KOBİ'lerine hitap eden, modüler SaaS platformu. Her müşteri ihtiyacına göre modül seçiyor, kendi veritabanına sahip oluyor.

🏢

Multi-Tenant SaaS

Her müşteri (tenant) kendi izole veritabanına sahip. Veriler hiçbir zaman karışmaz. Tenant'a özel DB provisioning otomatik yapılır.

🧩

Modüler Yapı

E-Ticaret, Finans, Web Builder, CRM, HR gibi modüller bağımsız .NET projeleri. Müşteri sadece aldığı modülleri kullanır.

🏪

Storefront

Her E-Ticaret tenant'ı kendi müşterilerine açık bir storefront (online mağaza) alır. Tema editörü ile görsel özelleştirme yapabilirler.

CQRS Pattern — Örnek Kod

Command (Yazma)
public record CreateProductCommand(string Name, decimal Price)
    : IRequest<Guid>;

public class CreateProductCommandHandler
    : IRequestHandler<CreateProductCommand, Guid>
{
    private readonly ECommerceDbContext _db;

    public CreateProductCommandHandler(ECommerceDbContext db)
        => _db = db;

    public async Task<Guid> Handle(
        CreateProductCommand request,
        CancellationToken ct)
    {
        var product = new WixiProduct
        {
            Id    = Guid.NewGuid(),
            Name  = request.Name,
            Price = request.Price,
        };
        _db.Products.Add(product);
        await _db.SaveChangesAsync(ct);
        return product.Id;
    }
}
Query (Okuma)
public record GetProductsQuery()
    : IRequest<List<ProductDto>>;

public class GetProductsQueryHandler
    : IRequestHandler<GetProductsQuery, List<ProductDto>>
{
    private readonly ECommerceDbContext _db;

    public GetProductsQueryHandler(ECommerceDbContext db)
        => _db = db;

    public async Task<List<ProductDto>> Handle(
        GetProductsQuery request,
        CancellationToken ct)
    {
        return await _db.Products
            .Select(p => new ProductDto(p.Id, p.Name, p.Price))
            .ToListAsync(ct);
    }
}
04 — Frontend

Frontend Klasör Yapısı

Feature-Sliced Design (FSD) katmanlı mimarisi. Her katmanın net bir sorumluluğu var, üst katman alt katmanı import eder.

app/
Router, AuthGuard, global provider'lar — iş mantığı YOK
pages/
Route bileşenleri — admin/, store-admin/, ecommerce/, storefront/, auth/ alt klasörlerine ayrılmış
widgets/
Birden fazla entity/feature kullanan layout bileşenleri — Sidebar, Header
features/
Form handler, iş akışı, POST/PUT/DELETE çağrıları — ThemeBuilder, Auth, TenantSelector
entities/
TS interface, Zustand store, sadece GET çağrıları — User modeli
shared/
Domain-agnostik UI bileşenleri, axiosConfig, yardımcı fonksiyonlar

CQRS Handler'ları Yaz

Her işlem için ayrı Command veya Query + Handler çifti oluştur. Birden fazla işlemi tek handler'a koyma.

// Application/Contacts/Commands/CreateContact/CreateContactCommand.cs
public record CreateContactCommand(string FullName, string? Email)
    : IRequest<Guid>;

// Application/Contacts/Commands/CreateContact/CreateContactCommandHandler.cs
public class CreateContactCommandHandler(CRMDbContext db)
    : IRequestHandler<CreateContactCommand, Guid>
{
    public async Task<Guid> Handle(CreateContactCommand req, CancellationToken ct)
    {
        var contact = new WixiContact { Id = Guid.NewGuid(), FullName = req.FullName, Email = req.Email };
        db.Contacts.Add(contact);
        await db.SaveChangesAsync(ct);
        return contact.Id;
    }
}
5

Module Extensions — DI Kaydı

// CRMModuleExtensions.cs
public static class CRMModuleExtensions
{
    public static IServiceCollection AddCRMModule(
        this IServiceCollection services, string connectionString)
    {
        services.AddDbContext<CRMDbContext>(o =>
            o.UseSqlServer(connectionString));

        services.AddMediatR(cfg =>
            cfg.RegisterServicesFromAssembly(typeof(CRMModuleExtensions).Assembly));

        return services;
    }
}

Sonra Program.cs'e ekle:

builder.Services.AddCRMModule(builder.Configuration.GetConnectionString("DefaultConnection")!);

[Authorize(Roles = "SuperAdmin,Admin")]

Frontend — Service Dosyası

// shared/api/services/crmService.ts
import { apiClient } from '../axiosConfig';

export interface ContactDto {
  id: string;
  fullName: string;
  email?: string;
  phone?: string;
}

export const crmService = {
  getContacts: () =>
    apiClient.get<ContactDto[]>('/crm/contacts').then(r => r.data),

  createContact: (data: Partial<ContactDto>) =>
    apiClient.post('/crm/contacts', data).then(r => r.data),
};

07 — Veritabanı

Multi-Tenant Veritabanı Stratejisi

Master DB tüm platformu yönetir. Her tenant kendi izole DB'sini alır.

Master DB — Wixi_App / Wixi_App_TEST

Platform genelinde paylaşılan veriler burada tutulur.

  • WixiUsers, WixiRoles, WixiPermissions
  • WixiTenants, WixiTenantSubscriptions
  • WixiModules, WixiModuleMenus
  • WixiMenus, WixiLanguages
  • WixiMailTemplates, WixiAuditLogs
Tenant DB — Her müşteri için ayrı

Tenant kayıt olduğunda provisioner otomatik oluşturur.

  • Products, Categories, Brands, Orders
  • Customers, Addresses
  • Pages, ThemeVersions
  • Contacts, Deals (CRM)
  • FinanceTransactions, Budgets

08 — CI/CD

Branch Stratejisi & Deploy

GitHub Actions ile otomatik build/test/deploy. Hiçbir şey lokalde deploy edilmez.

🌿
